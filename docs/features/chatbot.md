# Chatbot Feature

## Overview

The chatbot provides AI-powered assistance to students using course documents and materials.

## Features

### Document Processing
- PDF and DOCX file parsing
- Text extraction and indexing
- Document storage and retrieval

### AI Responses
- Context-aware responses based on course content
- Natural language understanding
- Multi-turn conversations

### Text-to-Speech
- Audio output for responses
- Multiple TTS engines supported
- Customizable voice settings

## Usage

Students can interact with the chatbot through the course interface to get help with course materials and questions.
