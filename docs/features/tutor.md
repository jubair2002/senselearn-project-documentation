# Tutor Features

## Dashboard
- Manage courses
- View student progress
- Access analytics

## Course Management
- Create and edit courses
- Add course modules
- Upload course materials (PDF, DOCX)
- Set target disability types
- Manage course enrollment

## Quiz Management
- Create quizzes
- Add questions and answers
- Set quiz parameters
- View student submissions

## Profile Management
- Update tutor profile
- Upload documents
- Manage verification status
