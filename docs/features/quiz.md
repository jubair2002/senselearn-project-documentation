# Quiz System

## Overview

The quiz system allows tutors to create quizzes and students to take them.

## Tutor Features

### Quiz Creation
- Create multiple choice questions
- Set correct answers
- Configure quiz parameters
- Assign quizzes to courses

### Quiz Management
- Edit existing quizzes
- View student submissions
- Review student performance

## Student Features

### Taking Quizzes
- Access assigned quizzes
- Answer questions
- Submit quiz responses
- View results

### Quiz Results
- See correct/incorrect answers
- View score
- Review feedback
